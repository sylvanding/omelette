# 功能

## 文献管道

### 关键词
- 三级层次结构（L1 主词，L2 子词，L3 具体词）
- LLM 扩展与相关概念发现
- 自动生成 WOS、Scopus、PubMed 检索式

### 多源检索
- Semantic Scholar、OpenAlex、arXiv、Crossref 联合检索
- 标准化元数据与 DOI 匹配
- 批量导入与冲突检测

### 去重
- 三阶段：DOI 精确匹配 → 标题相似度 → LLM 验证
- 并排比较，保留/跳过/替换操作
- 批量 LLM 自动解决

### 爬虫
- 多渠道：Unpaywall → arXiv → 直链
- 每篇论文状态追踪

### OCR
- MinerU 处理原生数字 PDF
- PaddleOCR 处理扫描件
- GPU 加速与 TTL 模型管理

### 订阅
- RSS 和 API 定时更新
- 可配置频率（每日/每周/每月）

## AI 与知识

### RAG 知识库
- LlamaIndex + ChromaDB 向量存储
- BAAI/bge-m3 GPU 嵌入
- bge-reranker-v2-m3 重排序

### Chat Playground
- ChatGPT 风格流式对话
- 工具模式：问答、引用查找、综述大纲、缺口分析
- 对话历史管理

### 写作助手
- 摘要、引用格式化、综述大纲
- 缺口分析（创新性与可行性评分）
- 多格式引用导出

## 研究工具

### 分析仪表盘
- 阅读进度分析
- 每周阅读柱状图
- 热门期刊列表
- GitHub 风格阅读热力图

### 趋势分析
- 年度发表量
- 引用趋势
- 新兴/衰退主题检测

### 作者网络
- d3-force 合作网络图
- 中心性指标与过滤

### 论文对比
- 并排摘要、元数据、引用对比
- 支持 2-5 篇论文

### 引用工具
- APA、MLA、Chicago、IEEE、GB/T 7714 格式
- 参考文献生成器

### 版本追踪
- Semantic Scholar 版本轮询
- 差异生成与升级保留

## 协作与体验

- **团队管理**：邮件邀请与 RBAC 权限
- **API 密钥**：SHA-256 哈希，作用域控制
- **收藏集**：自定义分组与 AI 标签建议
- **国际化**：完整中英文双语
- **PWA**：可安装，离线缓存
